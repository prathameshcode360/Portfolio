# Simple Portfolio Website

A simple and responsive **Single Page Portfolio Website** built using **HTML, CSS, and JavaScript**. This project is designed for beginners to understand the basic structure of a web application, semantic HTML, responsive layouts, and fundamental JavaScript concepts.

## 📌 Project Objective

The primary goal of this project is to learn the fundamentals of front-end web development by building a basic portfolio website from scratch without using any frameworks or libraries.

Through this project, you will understand:

- Semantic HTML structure
- Different sections of a webpage
- CSS layouts using Flexbox and Grid
- Responsive web design with Media Queries
- Basic DOM Manipulation
- JavaScript Event Handling
- Smooth scrolling and simple animations

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/prathameshcode360/Portfolio.git
```

2. Navigate to the project folder.

```bash
cd portfolio
```

3. Open `index.html` in your preferred web browser.

No installation or additional dependencies are required.
